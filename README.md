# 🦑 Sub Rush – The Golden Quest

A 2D game built in Python using Pygame, with a focus on game logic, interaction, and iterative improvement.

During development, I tested core mechanics, identified issues, and refined gameplay to improve clarity and user experience.

## 🎮 Gameplay

The player controls a submarine, collects coins, and avoids obstacles.

The goal is to reach 100 points while managing lives. As the score increases, the game becomes more difficult.

## 🚀 How to run

1. Download or clone the repository

2. Install Pygame:
   ```bash
   pip install pygame
   ```
   
3. Run the game:
   ```bash
   python sub_rush.py
   ```
**Note:** Run the script from the project root folder so asset paths work correctly.
